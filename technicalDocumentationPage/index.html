Show Grid

A grid option allows the example potholder to be divided up into squares.

The grid lines can be toggled by:

  • Clicking the Show Grid checkbox to display the grid lines in the example potholder box
  • Clicking the Show Grid checkbox again to turn off the grid lines in the example potholder box

          function onShowGridToggle(evt) {
            potholderGenerator.toggleGrid(evt.target.checked);
          }
Number of Colors

A number selector changes the number of colors displayed in the example potholder.

The number of potholder colors can be set by:

  • Increasing the Number of Colors to allow for the selection of more potholder colors
  • Decreasing the Number of Colors to reduce the selection of potholder colors

          function onSetNumColors(evt) {
            let numColors = evt.target.value;
          }
Pattern

The potholder pattern design can be changed by clicking an item in the Pattern selection box. This will change the pattern design in the example potholder box.

For unique patterns, a second pattern selection box will appear. This will save each unique pattern generated, in order to redisplay it later.


          patternSelector.init();
Colors

The potholder colors can be set by clicking a color circle. This will display a color palette.

Selecting a color from the color palette and clicking the Ok button will add the color to the example potholder box.

function onSelectColor(evt) {
            let options = {
                mouseX: evt.clientX,
                mouseY: evt.clientY
            };
            colorPalette.show(evt.target, options);
        }
Design Your Own

Custom patterns can be created by manually changing each color individually.

Each potholder color can be set manually by:

  • Clicking the Design Your Own checkbox to allow manual setting of each potholder row and column color to create your own pattern
  • Clicking the Design Your Own checkbox again to revert the example potholder back to the currently selected predefined pattern

          function onManuallySelectColors(evt) {
            initColorSelectors(MAX_COLORS);
          }
